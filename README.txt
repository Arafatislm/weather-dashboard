# weather-dashboard
Weather Dashboard is a lightweight, performance-optimized WordPress plugin that displays real-time weather for multiple cities using the Open-Meteo API. Features include caching, REST-based lazy loading, skeleton UI, animated icons, and full admin customization via shortcode.
